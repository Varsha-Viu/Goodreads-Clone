# 📚 Goodreads Clone 

A college final year project, this full-stack Goodreads Clone is built with Angular for the frontend and C# Web API for the backend. It showcases skills in web development, database management, API integration, and responsive UI design.

<br>

## 🚀 Features

* 📖 User authentication & profile management

* 📚 Book listing, reviews, and ratings

* 🔍 Search & filter books

* 🌟 Add books to favorites or reading lists

* 📝 Write and manage book reviews

<br>

## 🛠️ Tech Stack

### Frontend (Angular)

* Angular

* TypeScript

* Tailwind

* RxJS

### Backend (C# Web API)

* .NET Core

* Entity Framework

* SQL Server

## 💻 Setup Instructions

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/goodreads-clone.git
cd goodreads-clone
```
***
### 2️⃣ Frontend Setup (Angular)

```bash
cd goodreadsUI
npm install       # Install dependencies
ng serve         # Run Angular app
```

Open your browser and visit: http://localhost:4200
***
### 3️⃣ Backend Setup (C# Web API)

* Open goodreadsAPI in Visual Studio

* Restore dependencies:
    * Run dotnet restore

* Update appsettings.json with your database configuration

* Run the project
    * Press F5 (or dotnet run)

API should now be running at: https://localhost:5001

<br>

## ⚙️ Environment Variables
Create a .env or modify appsettings.json for API configuration.

## 📝 Useful Commands

### Frontend (Angular)
* ng serve → Start the development server

* ng build → Build for production

* ng test → Run unit tests

### Backend (C# Web API)

* dotnet run → Start the API

* dotnet build → Build the project

* dotnet test → Run tests
